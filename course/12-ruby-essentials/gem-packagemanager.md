# Understanding gem

gem is a package manager for ruby. And ruby-bundler is a cli tool which takes the Gemfile and install all the required packages.

# Example (gem):
Syntax:
```
gem install <package_name>
```

# Example:
Sytax:
```
bundle install
```

## `Gemfile` syntax:

```
source 'https://rubygems.org'

gem 'berkshelf
```
